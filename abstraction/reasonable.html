Reasonableness

Definition

Suppose p is a process on X. Then an element y is reasonable with respect to p if there exists x ∈ X s.t. p(x)=y. If there does not exist an element x such that p(x) = y, then y is unreasonable with respect to p.

This is deliberately general terminology: For any object or element x, and any process p, either x is reasonable with respect to p, or it is not. This language is intended to facilitate intelligent discussion about thoughts.

Example 1

For the successor function over the positive integers, all positive integers except 1 are reasonable.
