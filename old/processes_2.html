The Process

This paper introduces the process, a function without fixed points. Processes will form the mathematical backbone of this theory.

Definition

Suppose $X$ is a nonempty set. Then a function $p:X→X$ is a process if for all $x ∈ X, p(x) ≠ x$.

We might also say that $p$ is a process on $X$.

Example 1: Successor Function

Let $S:ℕ→ℕ$ by, $$ S(n) = n+1.$$ Then $S$ is a process.

Example 2: On a Finite Set

Let $p$ be a function on $\set{0, 1}$ by, $$ p(x) = \begin{cases} \ 1 \text{ if } x = 0, \\ \ 0 \text{ if } x = 1. \\ \end{cases} $$ Then $p$ is a process.

Consciousness

It is not sufficient to claim that a function without fixed points is conscious. For, if you were to imagine an actor that meets this definition, it would be a common robot (and, more importantly, one without memory).

Definition: If a process $P$ is conscious, then $P=P_n$ for some $n\in\mathbb{Z}$, and there exists a process $C$ such that $C(P) = P_{n+1}$. The process $C$ will be referred to as the consciousness of $P$.

Consciousness, for us, is not an intuitively difficult concept: A conscious process changes itself.

Realizations of Processes

In virtue of its mathematical definition, a process is a set. Of that set, we will typically be concerned with a single element of it.

Definition: A realization of a process $p$ is an element of $p$.

A realization is an ordered pair whose elements are distinct. If $p$ is a process, and $a$ is an element in its domain, then $(a,p(a))$ is a realization of p. In this case, $(a,p(a))$ has argument $a$ and result $p(a)$.

A Realization of a Process

Intuitively, a realization is a change.

Example 3: Successor Function

As above, let $S:ℕ→ℕ$ by, $$ S(n) = n+1.$$ Then $(2,3)$ is a realization of $S$.

Example 4: On a Finite Set

As above, let $p$ be a function on $\set{0, 1}$ by, $$ p(x) = \begin{cases} \ 1 \text{ if } x = 0, \\ \ 0 \text{ if } x = 1. \\ \end{cases} $$ Then $(0,1)$ and $(1,0)$ are the realizations of $p$.
