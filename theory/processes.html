The Process

This paper introduces the process, a function without fixed points. Processes are the (future) mathematical backbone of the theory.

Definition

Suppose $X$ is a nonempty set. Then a function $p:X→X$ is a process if for all $x ∈ X, p(x) ≠ x$.

We might also say that $p$ is a process on $X$.

Example 1: Successor Function

Probably the simplest mathematical process we can consider is the successor function: $$S:ℕ→ℕ \text{ by } S(n) = n+1$$

Example 2: On a Finite Set

A process need not be divergent. Let $p$ be a function on $\set{0, 1}$ by $$ p(x) = \begin{cases} \ 1 \text{ if } x = 0, \\ \ 0 \text{ if } x = 1. \\ \end{cases} $$ Then $p$ is a process.

Realized Process

In virtue of its mathematical definition, a process is a set. Of that set, we will typically be concerned with a single element of it. These we will call realized processes.

Each realized process has two discernible elements, Argument and Result:

The Realized Process

If $p$ is a process, and $a$ is an element in its domain, then $(a,p(a))$ is a realized process with argument $a$ and result $p(a)$.

A realized process is a change.
