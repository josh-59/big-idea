Long Term Memory

Long term memory is not the main focus of this theory, but is within the purview of it. Here, a plausible method of implementing it is presented. We will refer to the contents of the current memory as the context of the intellect.

  1. The context is hashed, and this hash is used as the name of a file
  2. To retrieve long-term memory, the file's contents are written to an arbitrary, fixed location in the context
  3. To store long-term memory, the context is reduced and saved to the file.

There are a few key points to this implementation:

  1. Long term memory is handled concurrently and unconsciously. It is not an explicitly controlled process
  2. It becomes the role of the intellect to "game" this system, in attempts to recall specific memories
  3. The process is self-reinforcing
  4. Memories can imply more memories: When recalling, the hash may yield a related memory.
  5. Mistakes do happen

A minor point also exists: As the intellect organizes, the form of memories would change. That is, early (childhood) memories would exist, but not in the form of the current intellect.
