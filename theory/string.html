String

Here, our goal is to describe the pattern of actions the active mind undertakes.

Definition: A string is a sequence of non-discrete threads.

We do not require that a string be finite, nor infinite, in length; nor do we preclude the possibility that it forms a loop or other structure. The only quality that we demand is that the threads are not separate, but overlap in time.
