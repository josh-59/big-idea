Intellect

My target object is the intellect, a self-conscious, learning process of order n > 0 and depth k > 3, I believe. I think that the mind is, effectively, a data structure. Specifically, it is a data structure which is endlessly economizing.

Moreover, I think that it is a data structure which operates in O(0) (Big-Oh-Zero) time; that is, it is infinitely economizing. As a conjecture, I think that any data structure operating in O(0) time will prove to be an intellect (and thus be an "AI" as we think of them).

A single abstraction has distinct inputs and outputs; it begins at one point and ends at another. An intellect does not. In particular, an intellect would observe the results of its own output (or, put the other way around, would interact with its input).
